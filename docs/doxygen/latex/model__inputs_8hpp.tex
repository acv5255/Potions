class \textbf{ Chem\+File}
\begin{DoxyCompactList}\small\item\em Chemical system parameters The class that contains the complete information about the primary species. This file contains the list of primary species including the total concentrations for each primary species. The Primary\+Species block requires both the name of the primary species and the total concentration. This section also requires the secondary species block to be populated with at least 1 secondary species. For a kinetic simulation, this requires the Mineral\+Kinetic block to have at least one mineral entry which contains both a mineral name and a mineral surface area. \end{DoxyCompactList}\item 
struct \textbf{ Secondary\+Species}
\begin{DoxyCompactList}\small\item\em The secondary species information struct This data structure contains the information for how to represent a secondary species in terms of primary species. \end{DoxyCompactList}\item 
struct \textbf{ Mineral\+Species}
\begin{DoxyCompactList}\small\item\em Data structure containing information for a mineral species. \end{DoxyCompactList}\item 
class \textbf{ Database}
\begin{DoxyCompactList}\small\item\em Data structure for data contained in the chemical database, \textquotesingle{}cdbs.\+yaml\textquotesingle{} This data structure contains all of the chemical information that the user does not need to change. In the database, there is a list of primary species, secondary species, including the equilibrium parameters, and mineral species including the kinetic and equilibrium parameters. \end{DoxyCompactList}\item 
class \textbf{ Model\+Inputs}
\begin{DoxyCompactList}\small\item\em Wrapper for the model inputs Data structure containing the input data to run the simulations. This is a way to interface with both the \doxyref{Chem\+File}{p.}{class_chem_file} and \doxyref{Database}{p.}{class_database} objects to simplify the model interface, as this class contains methods to construct the mass conservation, equilibrium, and kinetic constants. \end{DoxyCompactList}\end{DoxyCompactItemize}
