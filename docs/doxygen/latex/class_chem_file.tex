Chemical system parameters The class that contains the complete information about the primary species. This file contains the list of primary species including the total concentrations for each primary species. The Primary\+Species block requires both the name of the primary species and the total concentration. This section also requires the secondary species block to be populated with at least 1 secondary species. For a kinetic simulation, this requires the Mineral\+Kinetic block to have at least one mineral entry which contains both a mineral name and a mineral surface area.
