Construct the mass conservation matrix. 

This function constructs the mass conservation matrix from the list of primary species in this model. This matrix represents how each of the secondary species relate to the primary species. This model does not explicitly track H+ and instead uses a charge balance for all species for the entire solution. This allows the system to have closure while also imposing another physical law on the system. \begin{DoxyReturn}{Returns}
